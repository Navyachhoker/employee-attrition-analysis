# Employee Attrition Analysis

## Project Overview

This project focuses on analyzing employee attrition data using Python and data visualization techniques. The goal is to understand the factors affecting employee attrition and identify important relationships between employee attributes.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Jupyter Notebook

---

## Dataset Features

The dataset contains:

* Age
* Daily Rate
* Job Satisfaction
* Work Life Balance
* Years at Company
* Distance From Home
* Total Working Years
* Relationship Satisfaction
* Years With Current Manager

---

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Missing Value Handling
4. Duplicate Data Handling
5. Exploratory Data Analysis
6. Data Visualization
7. Relationship Analysis

---

## Visualizations Included

* Age vs Total Working Years
* Age vs Daily Rate
* Daily Rate vs Job Satisfaction
* Years at Company vs Job Satisfaction
* Work Life Balance vs Job Satisfaction
* Relationship Satisfaction vs Years With Current Manager

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/employee-attrition-analysis.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Jupyter Notebook:

```bash
jupyter notebook
```
